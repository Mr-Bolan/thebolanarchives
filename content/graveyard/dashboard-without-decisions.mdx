# graveyard_001 // dashboard_without_decisions

## why it died

The dashboard showed movement, but it did not change the next action. It made activity visible without making consequence visible.

## what was kept

The useful remnant is the question underneath it: what would make a display operationally decisive rather than merely informative?

## replacement thread

The replacement is not another dashboard yet. It is the smaller data trust ladder experiment, because the ladder names the missing states before pretending to instrument them.
