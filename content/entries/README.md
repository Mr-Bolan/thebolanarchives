# content/entries/

Longer archive articles. Content type: `entry`. Route: `/entries`.

These may be finished or semi-finished. Use the full post structure from brief section 15
(what this is / why it exists / what i built / what worked / what broke / what i understand now /
open questions / related artefacts).

Maturity statuses common here: `working_note`, `field_tested`, `stable_artefact`.

Naming: `slug.mdx` (e.g. `why-this-exists.mdx`). `README.md` is ignored by the loader.
