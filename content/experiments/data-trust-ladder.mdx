# experiment_001 // data_trust_ladder

## question

What changes when a dashboard moves from showing that data exists to showing that the data can support action?

## static fallback

The ladder is deliberately simple. Each step can fail independently, and a dashboard can look impressive while still being stuck on the first rung.
