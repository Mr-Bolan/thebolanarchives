# content/experiments/

Interactive technical demos, code experiments, prototypes, simulations, and visual
explanations. Content type: `experiment`. Route: `/experiments`.

This is where the site can evolve into an explorable-explanations platform. Posts here may
embed React islands wrapped in `ExperimentFrame` (see `src/components/experiments/`).

Maturity statuses common here: `sketch`, `working_note`, `field_tested`.

Naming: `slug.mdx`. `README.md` is ignored by the loader.
