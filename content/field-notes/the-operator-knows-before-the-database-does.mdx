## what this is

An observation about the gap between measured data and lived system awareness.

## what i noticed

The operator usually knows something is wrong before the database does. Not because the operator is mystical. Because the operator is watching a richer signal set.

## what it suggests

The archive should treat operational knowledge as evidence, not decoration around the evidence.
