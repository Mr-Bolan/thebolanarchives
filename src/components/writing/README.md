# src/components/writing/

Reading-experience components used inside MDX posts.

- `CodeBlock` — styled code block with optional filename + language label.
- `TerminalBlock` — CLI snippets and log-like output.
- `DiagramBlock` — container for diagrams, images, SVGs, future interactive modules.
- `SideNote` / `FootnoteRail` — margin notes, definitions, warnings, commentary.
- `TableOfContents` — for longer entries.
- `OpenQuestions` — displays unresolved questions.

Pending: all component code. Prioritise readability.
