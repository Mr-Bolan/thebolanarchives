# src/styles/

Global styling and design tokens. Dark, quiet, archival (brief sections 17-18).

- `tokens.css` — CSS variables: background, text, accents, borders (values in brief section 17).
- `global.css` — base resets and element defaults.
- `typography.css` — body (readable serif/humanist sans) + metadata (monospace) pairing.
- `archive.css` — archive-card / index / metadata styling.

Do not look like a generic Tailwind SaaS page. Pending: all stylesheets.
