# public/

Static assets served as-is. If a real custom domain is chosen, this is also where `CNAME`
will live to point GitHub Pages at that domain.

- `images/` — photos, screenshots, illustrations.
- `diagrams/` — diagrams and SVGs referenced by posts.
- `textures/` — subtle background textures (faint grid, paper grain, blueprint lines, scanlines).

Pending: `favicon` and assets. Add `CNAME` only when the real domain is known; do not commit
placeholder domains.
